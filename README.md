# ⚡ LightningChat

**LightningChat ⚡** is an ultra-fast AI chatbot powered by Groq LLMs,
built using Streamlit and LangChain.\
It delivers real-time conversational responses with a clean
ChatGPT-style interface.

------------------------------------------------------------------------

## 🚀 Features

-   ⚡ Ultra-fast responses using Groq (LLaMA 3.1 models)
-   💬 ChatGPT-style UI with chat history
-   🎛️ Adjustable parameters (temperature, max tokens)
-   🧠 Prompt-based conversational AI
-   🧹 Clear chat functionality
-   🎨 Clean and modern interface

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   **Frontend:** Streamlit\
-   **LLM:** Groq (LLaMA 3.1)\
-   **Framework:** LangChain\
-   **Language:** Python

## ⚙️ Installation

### 1. Clone the repository

``` bash
git clone https://github.com/your-username/lightningchat.git
cd lightningchat
```

### 2. Create virtual environment

``` bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

``` bash
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 🔑 Setup

1.  Get your Groq API key from: https://console.groq.com/
2.  Run the app:

``` bash
streamlit run app.py
```

3.  Enter your API key in the sidebar
